# sns_topic

Provides an SNS topic resource.

## input variables

| Name | Description | Type | Default | Required |
|------|-------------|:----:|:-----:|:-----:|
|sns_topic_name|The friendly name for the SNS topic.|string|{{ name }}|No|

## output parameters

| Name | Description | Type |
|------|-------------|:----:|
|sns_topic_arn|The ARN of the SNS topic|string|
