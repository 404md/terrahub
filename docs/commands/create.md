# terrahub create

## Help Message

```
Usage: terrahub create [options]

terrahub@0.0.1 (built: Mon Apr 7 12:00:00 EDT 2018)
this command will create terraform code from predefined templates

Options:
  --name, -n 		 Uniquely identifiable cloud resource name
  --template, -t 	 Template name (e.g. cloudfront, dynamodb, lambda, s3)
  --directory, -d 	 Path where template should be created (default: .terrahub)
  --parent, -p 		 Parent component path
  --force, -f 		 Replace directory
  --env, -e 		 Workspace environment
  --help, -h 		 Show list of available commands
```
