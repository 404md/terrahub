# terrahub workspace

```
Usage: terrahub workspace [options]

terrahub@0.0.1 (built: Mon Apr 7 12:00:00 EDT 2018)
this command will run `terraform workspace` across multiple terraform scripts

Options:
  --delete, -d 		 Delete workspace environment (paired with --env)
  --include, -i 	 List of components to include
  --exclude, -x 	 List of components to exclude
  --var, -r 		 Variable(s) to be used by terraform
  --var-file, -l 	 Variable file(s) to be used by terraform
  --env, -e 		 Workspace environment
  --help, -h 		 Show list of available commands
```
